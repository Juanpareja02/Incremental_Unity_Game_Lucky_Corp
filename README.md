# CoinRoomIncremental

Prototype for Coin Vortex: The Slot Machine Room.

Unity FPS incremental game where a central machine spits physical coins, the player collects them manually, buys machine upgrades, risks money in a simple casino action, and tries to unlock the door before the run timer ends.

## Current scope

- Physical coin spawner with click bursts and auto-spit upgrades.
- Manual raycast pickup.
- Economy HUD.
- Upgrade shop.
- Double-or-nothing casino row.
- Run timer, coin goal, unlockable door, win/lose state.
- Runtime UI polish for HUD, progress bar, shop overlay, row styling and crosshair.

## Unity

Project version: Unity 6000.2.6f2.
